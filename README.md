# Space Odyssey – Interactive Educational Platform

## Project Overview  
**Space Odyssey** is an interactive, educational web application dedicated to space exploration and historic space missions. Designed with user engagement in mind, the platform leverages dynamic **JavaScript** functionalities to transform the learning experience of space science into an intuitive and immersive journey.

---

## Site Structure  
The application consists of the following core pages:  

- **Home (`index.html`)**  
  Introduces the project, its purpose, and featured space missions.  

- **Missions (`missions.html`)**  
  Displays a comprehensive list of space missions with full CRUD capabilities (Create, Read, Update, Delete), dynamic search and filtering, and a favorites management system.  

- **Contact (`contact.html`)**  
  Includes a user-friendly contact form with real-time, client-side validation and clear error messaging.

---

## Key Features  

### 1. Form Validation  
- Ensures all required fields are completed.  
- Validates email format using robust pattern matching.  
- Provides immediate, user-friendly error messages below invalid fields.

### 2. Dynamic Search & Filtering  
- Live search functionality to filter missions by name.  
- Advanced filtering options by space agency, mission year, and mission type.

### 3. Full CRUD Operations  
- Users can **create**, **view**, **edit**, and **delete** mission entries directly through the interface.  
- All changes are reflected instantly in the DOM without page reloads.

### 4. Favorites Management  
- Toggle favorite status for any mission with a dedicated button.  
- A dedicated “Favorites” section displays only user-selected missions, persisted using browser local storage.

---

## Technologies Used  
- **HTML5**: Semantic structure and content organization  
- **CSS3**: Responsive, accessible, and visually cohesive design  
- **JavaScript (ES6+)**: Dynamic interactivity, DOM manipulation, form validation, CRUD logic, and local data persistence
